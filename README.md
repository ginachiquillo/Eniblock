# Eniblock – CTO Portfolio Context

## Overview

I served as **CTO of Eniblock**, a blockchain SaaS platform focused on **tokenization, digital asset infrastructure, and decentralized data collaboration**.

The company is no longer active, but its open-source components remain publicly available.

👉 Original repositories: https://github.com/eniblock

This repository provides context on the platform, architecture, and my role in its development.

---

## Platform Scope

Eniblock delivered a **full-stack blockchain platform** composed of multiple products:

* **Tokenize** – Security token issuance and lifecycle management platform
* **Secure** – Enterprise-grade digital asset custody (MPC, key recovery, multi-sig)
* **Connect** – Blockchain infrastructure orchestration (BaaS, APIs, monitoring)
* **Collaborate** – Decentralized data sharing and governance layer
* **Create** – NFT and marketplace infrastructure

These products enabled enterprises to **issue, manage, and exchange digital assets on public and private blockchains**.

---

## Technical Highlights

* Multi-blockchain support (Ethereum, Stellar, Bitcoin)
* Smart contract–driven business processes (tokenization, trading, settlement)
* **3-layer architecture**:

  * User applications (Investor / Issuer dashboards)
  * Smart contract layer (automation, token logic, governance)
  * Distributed ledger layer (data integrity & interoperability)
* Secure key management:

  * Multi-party computation (MPC)
  * Multi-signature workflows
  * Key recovery mechanisms
* Integration with enterprise systems (KYC, ERP, CMS)
* GDPR-aware data sharing and privacy design

## Production Use Cases
